# Sky Jewelry WebApp

Мини-приложение для Telegram WebApp: обложка, главное меню, подбор камня, каталог, справочник и форма индивидуального запроса.

## Команды
- `npm install`
- `npm run dev` — Vite dev server (по умолчанию 5173)
- `npm run build` — сборка в `dist/`
- `npm run preview` — предпросмотр собранной версии

## Переменные окружения
- `VITE_API_URL` — базовый URL API (например, `http://localhost:3000` в dev)
- `VITE_DEV_INIT_DATA` — тестовое значение `initData` для проверки без Telegram

По умолчанию, если приложение запущено на 5173, API ищется на `http://localhost:3000`.
